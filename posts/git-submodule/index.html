    Git Submodule

    发布于 2019-12-05 DRAFT

    使用 Git Submodule 注意事项。

    注意事项

    同名文件夹

    出现以下报错信息，说明本地已经存在同名文件夹，不能从 git 中拉取内容进行子模块的创建。

    需要将本地文件夹删除后（注意备份数据），重新运行命令即可。

    $ git submodule add git@github.com:xxx/xxx.git folder
# 'folder' already exists and is not a valid git repo

    远程仓库为空

    如果远程仓库是新创建且为空的，直接创建子模块也会提示错误。

    建议远程仓库创建时，同时新建 README.md

    $ git submodule add git@github.com:xxx/xxx.git folder

Cloning into '/xxx/folder'...
warning: You appear to have cloned an empty repository.
fatal: You are on a branch yet to be born
Unable to checkout submodule 'folder'
